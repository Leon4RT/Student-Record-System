# Student Record System

A full-featured Student Record System built with Python Flask and a modern HTML/CSS/JS frontend.

## Setup

1. Install dependencies:
   ```
   pip install flask
   ```

2. Run the application:
   ```
   python app.py
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:5000
   ```

## Features
- Add, view, update, and delete student records
- Subjects: CC 101, MATH 101, CC 100, GEd 102, PATHFit 1, NSTP, GEd 105, GEd 101
- College grading system (1.00 highest – 5.00 failed)
- GWA computed using unit-weighted average
- Remarks: Excellent / Superior / Very Good / Good / Passing / Failed
